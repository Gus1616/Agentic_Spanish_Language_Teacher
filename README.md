# Agentic_Spanish_Language_Teacher
Agentic Spanish Language Teacher
<img width="1132" height="775" alt="image" src="https://github.com/user-attachments/assets/74fd2a87-fd09-4683-a715-f64a74a175f2" />


The assessment agent asks level-testing questions and scores the answers.
The lesson agent takes a topic (e.g. "past tense") and generates a mini-lesson.
The practice agent creates exercises (fill-in-the-blank, translation, etc.) based on what was just taught.
The feedback agent receives the user's answer, grades it, and explains any errors in plain language.
